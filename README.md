# Ticket-Logging-System

## Application Overview
This application is a web-based **Service Desk / Ticket Logging System**, developed as part of my **Software Engineering & DevOps module assignment**.

The system allows users to raise support tickets, track their progress, and enables analysts and administrators to manage, update, and assign tickets. It simulates a real-world IT support environment using role-based access and structured workflows.

## Key Features

- Users can register new accounts and log into the system
- Users can create and manage their own tickets  
- Analysts can view, update, and assign tickets to other analysts  
- Admins can assign tickets, manage all tickets, and delete existing tickets and users  
- Notes can be added to tickets for updates and communication  
- Role-based permissions ensure correct access for each user type
